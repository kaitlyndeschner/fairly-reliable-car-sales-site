# Fairly Reliable Car Sales Site

A modern ASP.NET Core MVC web application for browsing, managing, and showcasing vehicle listings.

## Overview

Fairly Reliable Car Sales Site is a full-stack web application developed as a college project using ASP.NET Core MVC. The application allows users to browse available vehicles, view detailed car information, and interact with a clean, responsive user interface.

The project demonstrates:

* ASP.NET Core MVC architecture
* Entity Framework Core integration
* SQL database usage
* CRUD operations
* Razor Views
* Responsive frontend design
* Authentication and authorization concepts
* Clean project structure and maintainable code

## Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/kaitlyndeschner/fairly-reliable-car-sales-site.git
```

### 2. Navigate to the Project Folder

```bash
cd fairly-reliable-car-sales-site
```

### 3. Restore Dependencies

```bash
dotnet restore
```

### 4. Build the Project

```bash
dotnet build
```

### 5. Run the Application

```bash
dotnet run
```

The application will run locally on:

```text
http://localhost:5240
```
